# 运营看板指标格式设计

## 目标

将运营看板中的原始数值改为调度员可快速识别的业务格式，减少对 `1.0000`、`3.00` 等内部数值口径的解释成本。

## 展示规则

- 比例指标按百分比展示：`1.0000` 显示为 `100%`，`0.875` 显示为 `88%`。
- 等待时间和绕行时间统一显示为一位小数的分钟值：`3` 显示为 `3.0 分钟`。
- 订单量保持整数展示。
- 不修改后端指标计算、接口字段或数据库数据。

## 影响范围

- 仅修改企业运营端的“运营看板”页面及其展示层格式化函数。
- 调度工作台、订单中心、车辆任务和审计日志不改变既有数值展示。

## 异常处理

- 格式化函数仅接收接口定义的数值或数值字符串。
- 无法转换的值保留原始文本，避免把异常数据误显示为有效指标。

## 验收与测试

- 单元测试覆盖零值、普通比例、普通分钟值和不可转换值。
- 看板页面测试确认比例和时长以业务格式呈现。
- 前端类型检查、完整测试和生产构建通过。
